# TO DO with JSON

## What the Application Do:

1. Adds tasks
2. Marks the tasks "Done"
3. Modify tasks
4. Deletes tasks

## Usage Image

```
todo add task_title task_description
```

```
todo show
[Display the tasks some how]
```
